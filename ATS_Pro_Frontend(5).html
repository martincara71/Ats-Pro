📌 Guía de configuración paso a paso

1. Abrí script.google.com → Nuevo proyecto
2. Pegá el código del archivo BACKEND_Code.gs
3. Creá una Google Sheet nueva → copiá su ID (el código largo de la URL) y pegalo en la variable SPREADSHEET_ID
4. En el editor: Implementar → Nueva implementación → Aplicación web
   • Ejecutar como: Yo
   • Acceso: Cualquier persona
5. Copiá la URL de la implementación → pegala en el campo de arriba
6. ¡Listo! Todos los que tengan esta URL del frontend tendrán acceso a los mismos datos.

Para publicar el frontend: Subí el HTML a GitHub → activá GitHub Pages → compartí la URL pública.
